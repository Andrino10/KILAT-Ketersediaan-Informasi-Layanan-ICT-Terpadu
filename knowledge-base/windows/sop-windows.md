# SOP Troubleshooting - Divisi Windows

## Tidak Bisa Login Windows

### Penyebab yang Mungkin Terjadi
- Password salah atau Caps Lock aktif
- Akun pengguna terkunci (locked out)
- Profil pengguna rusak

### Langkah Penyelesaian
1. Pastikan tombol Caps Lock tidak aktif (perhatikan lampu indikator di keyboard)
2. Periksa apakah Anda mengetik password dengan benar, perhatikan huruf besar dan kecil
3. Jika ada pesan "Account Locked", tunggu 15-30 menit lalu coba lagi
4. Coba restart komputer terlebih dahulu, lalu login kembali
5. Jika masih tidak bisa login, kemungkinan perlu reset password oleh Engineer ICT

### Kategori
Masalah Ringan

## Laptop Lemot / Komputer Lambat

### Penyebab yang Mungkin Terjadi
- Terlalu banyak program yang berjalan bersamaan
- Penyimpanan (disk) hampir penuh
- Komputer sudah lama tidak di-restart
- Update Windows sedang berjalan di background

### Langkah Penyelesaian
1. Tutup program atau tab browser yang tidak digunakan
2. Restart komputer (pilih Restart, bukan Shut Down) untuk membersihkan memory
3. Periksa kapasitas penyimpanan: klik kanan pada drive C di File Explorer, pilih Properties, pastikan masih ada ruang kosong
4. Hapus file-file yang tidak diperlukan dari folder Downloads atau Recycle Bin
5. Tunggu beberapa saat jika ada notifikasi Windows Update sedang berjalan, setelah selesai komputer biasanya kembali normal

### Kategori
Masalah Ringan

---

## Aplikasi Tidak Bisa Dibuka / Error

### Penyebab yang Mungkin Terjadi
- Aplikasi crash atau hang
- File aplikasi rusak
- Konflik dengan update Windows terbaru

### Langkah Penyelesaian
1. Tutup aplikasi yang error menggunakan Task Manager (tekan Ctrl+Alt+Delete, pilih Task Manager)
2. Di Task Manager, cari aplikasi yang bermasalah, klik kanan, lalu pilih "End Task"
3. Tunggu beberapa detik, lalu buka kembali aplikasi tersebut
4. Jika masih error, coba restart komputer lalu buka kembali
5. Jika masalah berlanjut, kemungkinan aplikasi perlu di-install ulang oleh Engineer

### Kategori
Masalah Ringan
